Plugin de WordPress con patrones accesibles y child theme para crear sitios web modernos.

== Description ==

Aqua Site Kit es un plugin completo que incluye:

* Patrones de bloques accesibles y semánticos
* Child theme con theme.json configurado
* Sistema de tokens de color integrado
* CSS ligero usando variables de WordPress
* Compatibilidad total con Gutenberg

= Características principales =

* **Accesibilidad**: Cumple con estándares WCAG 2.1 AA
* **Semántica**: HTML semántico y estructurado
* **Gutenberg Nativo**: Usa bloques nativos de WordPress
* **Sistema de Tokens**: Colores brand-500/700 y neutral-100/900
* **CSS Ligero**: Variables CSS de WordPress, sin CSS pesado
* **Responsive**: Diseño adaptable a todos los dispositivos

= Patrones incluidos =

* Hero Section
* Call to Action
* Testimonials
* Features Grid
* Pricing Table
* Contact Form

= Child Theme =

Incluye un child theme de Twenty Twenty-Four con:
* theme.json configurado
* Sistema de tokens de color
* Estilos personalizados
* Compatibilidad total con el editor de bloques

== Installation ==

1. Sube el archivo del plugin a la carpeta `/wp-content/plugins/aqua-site-kit/`
2. Activa el plugin a través del menú 'Plugins' en WordPress
3. Los patrones estarán disponibles en el editor de bloques
4. El child theme estará disponible en Apariencia > Temas

== Frequently Asked Questions ==

= ¿Es compatible con todos los temas? =

Sí, los patrones funcionan con cualquier tema compatible con Gutenberg.

= ¿Requiere conocimientos técnicos? =

No, está diseñado para ser fácil de usar para cualquier usuario.

= ¿Es accesible? =

Sí, todos los patrones cumplen con estándares de accesibilidad WCAG 2.1 AA.
